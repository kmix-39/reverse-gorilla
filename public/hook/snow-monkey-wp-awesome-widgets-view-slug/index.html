snow_monkey_wp_awesome_widgets_view_<$slug>

フックタイプ

アクションフック

説明

WPAW ウィジェットの表示時に実行される処理を追加します。

補足説明

  • 本フックは、WPAW が接頭辞に付与されているウィジェットのみ実行されます。
  • フック名の <$slug> に当たる部分は、該当ウィジェットの slug を記述します。
  • WPAW ウィジェットの slug の値は、WPAW ウィジェットのslug値の一覧 を参照してください。
  • 該当ウィジェットに $name が存在する場合、本フックではなく snow_monkey_wp_awesome_widgets_view_<$slug>-<$name> の使用を推奨します。

引数

引数名タイプ説明
$name文字列（string）WPAW ウィジェットの name の値
$vars配列（array）WPAW ウィジェットの引数配列

サンプルコード

my-snow-monkey.php
<?php
  … 略 …

  add_action(
    'snow_monkey_wp_awesome_widgets_view_<$slug>',
    function( $name, $vars ) {

      … ここに「ウィジェットの表示時に実行される処理」を記述しましょう …

	  }
  );
